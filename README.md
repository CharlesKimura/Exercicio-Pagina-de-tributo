# Exerc-cio-P-gina-de-tributo
Projeto de Página de Tributo responsiva, construído com HTML semântico e CSS. Utiliza Flexbox para criar um layout de linha do tempo em duas colunas e @media queries para garantir a adaptação em diferentes telas, de desktops a celulares. Demonstra habilidades em estruturação, layout e design responsivo.
